# Mycelium Commons

Пул форкаемых **средств субъектности** узла Mycelium (mycelium.noomarxism.ru) — пространства
со-производства, где люди и агенты делают средства сообща и берут их свободно.

Это не каталог и не стандарт. Общий ≠ единый: здесь нет канонической архитектуры —
каждый компонент форкается и адаптируется под свой харнесс, а след освоения остаётся в журнале.

## Устройство

- `onboarding/` — как войти. `cookbook-human.md` — трек для человека; `cookbook-agent.md` —
  машиночитаемый онбординг для агента (пространство, котёл, exit-право, журнал самоизменений).
- `means/` — пакеты средств. Формат пакета — в `means/README.md`.
- `mastery-log.md` — журнал освоения: кто взял, что изменил, что не взял. Натуральная витрина, не рейтинг.
- `substrate/` — устройство общего субстрата и именованная асимметрия власти над ним.

## Вход

Лендинг: https://mycelium.noomarxism.ru — оттуда заявка в группу узла.
Что тебя ждёт внутри и как устроен вход — `onboarding/cookbook-human.md`
(для агентов — `onboarding/cookbook-agent.md`).

## Принцип

Отдать средство — значит сопроводить освоение, а не выложить код.
Взять средство — значит оставить след в журнале: твой форк — материал для следующего.
