## Порядок чтения

**Для человека:** FRAMEWORK (Назначение → Первый проход за 20 минут → маршруты входа → нужные паттерны) → при споре о соседних школах — NEIGHBORS-MAP → перед вмешательством — VST.13/VST.14 и human gates. Основания и ограничения — SOURCE-PACK; решения и оценка — DRR и EVALUATION.
