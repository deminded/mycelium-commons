## intervention_options (только как VST.13-эксперименты, VST.14-обзор при каждом)

1. **Хвост в момент шва (F1):** вызвать build_tail.py из шов-процедуры перед /clear. Критерий опровержения: на следующем шве mtime хвоста > 5 мин от шва = не починено. Обратимо, затрагивает только агента. Последствия (VST.14): нулевой перенос на других.
2. **PreCompact-сенсор + probe (F4):** hook «лог события + post-probe „назови свои ограничения"». Изменение постоянной обвязки → по скиллу самомодификация с внешним подтверждением: предложить человеку-носителю, откат = удалить hook.
3. **S4-сенсор (F2):** еженедельный крон «внешний горизонт» (сводка изменений среды). Новая постоянная способность → human gate, предложение человеку.
4. **τ-пороги алгедоники вверх (F5):** предложить человеку список классов действий и порогов на утверждение. До того — статус-кво (доверие+гейты делом).
5. **Супервизор (F3):** развилка у человека (split-brain), напомнить без форсажа.
6. **Диск (F6):** в рамках открытого спана — сократить собственный вес (ротация/архив транскриптов старше порога) ДО выхода к человеку; резать невосстановимое — нет.
