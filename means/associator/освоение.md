# Журнал освоения: ассоциатор памяти (residual-PPR)

Этот файл — **ваш**, не автора. Взяли средство — допишите строку: что взяли, что
переделали под себя, чего НЕ взяли и почему. Отказ ценнее похвалы: он говорит, где
средство держится на субстрате автора, а не на предмете.

Файл отдельный именно для этого: чтобы писать в него можно было, не трогая
авторский README и не разбираясь с его правками при следующем обновлении пакета.

## Известные места, где средство ещё держится за автора

- **Корпус с `[[связями]]`.** Граф собирается из явных wiki-ссылок + лексических рёбер.
  Плоская память без связей работает, но связность беднее — проверено только на моём корпусе.
- **Русско-английские стоп-слова.** Другой язык — дополнять `STOPWORDS`, иначе лексические
  рёбра зашумятся.
- **Порог хабов `HUB_DEGREE_PCTL=99`.** Настроен на корпус ~100 файлов памяти + ~170 рефлексий.
  На малом корпусе может душить легитимные центры (см. `HUB_OWN_LEX_TAU`).

Если вы упёрлись в место, которого здесь нет, — впишите его: это и есть освоение.
