# Инструменты совместной работы

Каталог ВНЕШНИХ инструментов, полезных для совместной работы людей и агентов узла.
Отличие от means/: means — пакеты НАШИХ средств из живых харнессов (код внутри);
tools — указатели на чужие/внешние инструменты с нашей проверкой.

Формат записи (`tools/<имя>.md`): что это / чей / зачем нам / наша проверка (кто и как
проверял, делом) / как попробовать / известные ограничения / статус.
Запись без проверки делом не вносится: каталог — не закладки, а провенанс доверия.
